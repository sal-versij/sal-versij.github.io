Refertazione

Accettazione

Creazione referto con immissione iniziale dei dati del paziente e del tipo di esame

Dati necessari

  • Data Accettazione (precompilata)
  • Anagrafe paziente
    • Nome
    • Cognome
    • Data di Nascita
    • Sesso
    • Luogo di Nascita
    • Codice Fiscale (generazione automatica + modificabilità)
    • Indirizzo + CAP + Citta + Provincia
  • Origine creazione referto(riferimento al centro)
  • Materiale in esame (Riferimento all'esame)

Campionamento

Inserimento Informazioni aggiuntive

Dati necessari

  • Macroscopia
  • Numero Block
  • Data (precompilata)

Diagnosi

Inserimento Diagnosi

Dati necessari

  • Data Diagnosi (precompilata)
  • Diagnosi
  • Medico (precompilato, ma selezionabile)
  • Pt + Pn + G (Non so se servono ancora ne so cosa sono)
  • NB

Listino

Calcolo del Listino Prezzi con eventuali sconti

Dati necessari

  • Sconto

Distribuzione e Stampa

Smistamento Diagnosi nei vari canali, con possibilità di stampa
